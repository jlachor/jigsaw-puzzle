# Jigsaw Puzzle

A browser-based jigsaw puzzle game. Upload any image, choose a grid size, and assemble the pieces.

## Install

```bash
npm install
```

## Run

```bash
npm run dev
```

Opens at `http://localhost:5173`.

## Build for production

```bash
npm run build
npm run preview
```

## How to play

1. **Upload** an image (any format)
2. **Adjust** the grid size with the column/row sliders
3. **Reshuffle** to randomize the piece shapes, then click **Start**
4. **Drag** pieces to move them — nearby neighbors snap together automatically
5. **Box select** — click and drag on empty space to draw a selection rectangle, then drag selected pieces together
6. **Zoom** — scroll wheel zooms in/out (centered on cursor)
7. **Pan** — middle-click drag or hold Space + left-click drag to move the view
